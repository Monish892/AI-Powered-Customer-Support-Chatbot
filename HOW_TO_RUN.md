# How to Run — AI-Powered Customer Support Chatbot

1. Set up a MySQL database and update the `.env` file with your credentials.
2. Install Python dependencies: `pip install -r requirements.txt`
3. Initialize the database by running the `schema.sql` script in your MySQL server.
4. Run the application: `python app/main.py`
5. Open your browser and navigate to `http://127.0.0.1:5000`.
