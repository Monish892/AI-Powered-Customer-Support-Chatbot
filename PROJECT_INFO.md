# Nexus AI - Powered Customer Support Chatbot

A senior-level web application demonstrating a Python and Flask-based chatbot using AIML for conversational AI, MySQL for data persistence, and a modern, responsive HTML/CSS/JS frontend. The project showcases a clear separation of concerns, a production-quality UI, and best practices in backend development.

## Tech Stack
- **Python** _(backend)_
- **Flask** _(backend)_
- **AIML** _(backend)_
- **MySQL** _(database)_
- **SQLAlchemy** _(backend)_
- **HTML** _(frontend)_
- **CSS** _(frontend)_
- **JavaScript** _(frontend)_

## Features
- AI-powered chat using AIML
- Real-time chat interface
- Conversation history storage in MySQL
- RESTful API for chat interaction
- Modern, responsive, dark-mode-ready UI
- Clean, modular, and scalable project structure
